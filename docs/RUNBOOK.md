# Runbook

## Goal
Ship one functioning single Android app suitable for Play Store sale.

## Roles
- Josko: product owner and approval authority
- ChatGPT: orchestrator, QA lead, decision control
- Claw: speed executor for implementation and document updates
- Ian: rapid design option generation
- Sentinels: evidence-based monitoring only

## Rules
- No architecture drift away from the single-app Play Store goal
- No feature is counted complete without observable evidence
- Claw output must be reviewable and testable
- Ian output must support quick product decisions
- Sentinel claims require logs, artifacts, or measurable signals

## Current focus
- stabilize BLE behaviour
- reduce server-side power drain
- converge toward single-app architecture
- maintain reproducible documentation
