# Presence Protocol Architecture Summary

## Overview
This document summarizes the architecture of the Presence Protocol, detailing its components and interactions.

## Key Components
- **BLE Handshake**: Mechanisms for device discovery and interaction.
- **Proof Generation**: Structure for encounters and token integrity.
- **Decentralization**: Interactions with Cardano blockchain and privacy layers.

## Architecture Goals
- Scalability
- Security
- Efficient resource usage
